## Setup Instructions

- In the `/web/src/common/firebaseConfigProperties.js` file, include valid firebase config keys.
- cd to `/web` folder.
- Run `npm install`.
- Build with `npm run build` command.
- For *dev server*, run `npm run dev`. Access app on localhost:8080.
